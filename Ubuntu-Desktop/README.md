## Detalhes da Configuração

- Principais programas utilizados foram:

  > Nautilus: explorador de arquivos
  > 
  > Terminal do Ubuntu: pode-se utilizar o Kitty ou outro de preferência
  > 
  > Chrome: instaladas as extensões do Vimium, uBlock e Hide Shorts for YouTube

- As extensions instaladas no Gnome Tweaks, foram:

  > App menu
  >
  > Blur my Shell
  > 
  > Quick settings tweaks
  >
  > Rounded window corners reborn
  >
  > RunCat
  > 
  > Vitals: configurado para mostrar a temperatura "average"; memória "usage"; processador "usage"
